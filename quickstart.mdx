## Comece em três passos

Configure sua integração com a API Verifica AI e comece a analisar contratos com inteligência artificial.

### Passo 1: Autenticação

<AccordionGroup>
  <Accordion icon="lock" title="Configure a autenticação Firebase">
    A API Verifica AI utiliza autenticação via Firebase. Você precisará:
    
    1. Configurar um projeto Firebase
    2. Obter o token de autenticação após o login
    3. Incluir o token no header `Authorization` de todas as requisições
    
    <Card
      title="Ver guia completo de autenticação"
      icon="lock"
      href="/essentials/authentication"
    >
      Aprenda como configurar e usar a autenticação
    </Card>
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Teste sua primeira requisição">
    Após configurar a autenticação, teste o endpoint de health check:
    
    ```bash
    curl https://api.verifica.ia.br/
    ```
    
    Você deve receber uma resposta confirmando que a API está funcionando.
    
    <Tip>Use o ambiente de desenvolvimento (`http://localhost:3333`) para testes locais.</Tip>
  </Accordion>
</AccordionGroup>

### Passo 2: Faça upload de um contrato

<AccordionGroup>
  <Accordion icon="file-upload" title="Upload de contrato">
    Faça upload do seu primeiro contrato:
    
    ```bash
    curl -X POST https://api.verifica.ia.br/contracts \
      -H "Authorization: Bearer SEU_TOKEN_FIREBASE" \
      -F "file=@/caminho/para/seu/contrato.pdf" \
      -F "name=Meu Contrato"
    ```
    
    Formatos aceitos: PDF, DOC, DOCX (máximo 10MB)
  </Accordion>
  <Accordion icon="list" title="Listar contratos">
    Liste todos os seus contratos:
    
    ```bash
    curl -X GET https://api.verifica.ia.br/contracts \
      -H "Authorization: Bearer SEU_TOKEN_FIREBASE"
    ```
  </Accordion>
</AccordionGroup>

### Passo 3: Analise um contrato

<Accordion icon="brain" title="Iniciar análise de risco">
  Inicie uma análise de risco para um contrato:
  
  ```bash
  curl -X POST https://api.verifica.ia.br/risk-analysis/contracts/{contractId}/analyze \
    -H "Authorization: Bearer SEU_TOKEN_FIREBASE"
  ```
  
  A análise é processada de forma assíncrona. Use o endpoint GET para verificar o status e obter os resultados.
</Accordion>
